What is CVP?

CVP is based on IVR technology. In this system we provide the institution with a unique phone number .The user can call to this number to know student information. The user makes his queries and based on the query the call is redirected to the student database and a response is generated. We use KOOKOO platform which is used to convert texted data to voice xml.
