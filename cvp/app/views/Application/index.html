COLLEGE VOICE PORTAL


The project college voice portal provides a IVR based information access system for colleges, schools, and other educational institutions.


Existing system:

Now a days many educational institutions are using short message services (sms) for day to day queries of the students and parents. But these systems are not reliable as messages may or may not reach the user in time. Also due to TRAI act and DND (do not disturb) service the instititutions are not able to send large no of messages on a single day.


The other major problem is the existing system is one way communication as the messages are generated by institutions to send information .But if the user have any queries he cannot communicate with the institution


Proposed system:

The proposed system is based on IVR technology. In this system we provide the institution with a unique phone number .The user can call to this number to know student information. The user makes his queries and based on the query the call is redirected to the student database and a response is generated. We use KOOKOO platform which is used to convert texted data to voice xml.
